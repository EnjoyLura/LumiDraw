Mini Program

小程序适配规范

针对微信小程序运行环境的专项设计适配指南，确保设计令牌在 rpx 单位体系下的精准还原。

设计稿基准
基于 750 x 1334 rpx 设计稿，1rpx = 0.5px（@375pt 宽度）。所有间距、字号均需换算为 rpx 单位，确保多设备等比缩放。
深色模式适配
通过 wx.getSystemInfoSync().theme 获取系统主题，配合 CSS 变量动态切换色彩令牌。
安全区域
底部导航栏需避开 Home Indicator 区域，使用 env(safe-area-inset-bottom) 适配全面屏。
性能约束
单页面 WXML 节点数建议不超过 1000，避免大面积 blur 滤镜和复杂阴影，确保 60fps 流畅渲染。推荐使用 CSS 变量而非行内样式。
